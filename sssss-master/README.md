# [DJANGO] SETTING

### REST FRAMEWORK

```BASH
$ pip install djangorestframework
```

### REST FRAMEWORK JWT

```BASH
$ pip install djangorestframework-jwt
```

### CORS-HEADERS

```BASH
$ pip install django-cors-headers
```

### DRF-YASG

```BASH
$ pip install -U drf-yasg
```
